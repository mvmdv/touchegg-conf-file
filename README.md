Swipe
- 3 fingers:
 - down:show desktop
 - up:show workspace
 - left-right: swicth desktop

- 4 fingers:
 - up:increase volume
 - down decrease volume

-Pinch
 - 2 fingers:
  zoom in and out
 - 3 fingers:
  Show Desktop Grid
